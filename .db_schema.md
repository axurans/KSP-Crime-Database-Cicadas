# **KSP Crime Database Schema Reference**

This document serves as the runtime schema context for the Router Agent to formulate syntactically correct ZCQL queries.

## **1. Core Tables**

### **Nodes**
Represents entities in the temporal intelligence graph (e.g., suspects, locations, phones).
*   `node_id` (Primary Key, VarChar, 64)
*   `name` (VarChar, 255) - Redacted in public queries.
*   `label` (VarChar, 50) - e.g., 'SUSPECT', 'TELECOM', 'BANK_ACCOUNT', 'LOCATION'
*   `created_at` (DateTime) - Timestamp when node was active.
*   `dissolved_at` (DateTime, Nullable) - Timestamp when node dissolved.

### **Edges**
Represents links and interactions between nodes.
*   `edge_id` (Primary Key, VarChar, 64)
*   `source_id` (VarChar, 64, Foreign Key -> Nodes.node_id)
*   `target_id` (VarChar, 64, Foreign Key -> Nodes.node_id)
*   `relationship_type` (VarChar, 50) - e.g., 'CALL_RECORD', 'CO_ACCUSED', 'FINANCIAL_TX'
*   `weight` (Double) - Correlation strength score (0.0 to 1.0)
*   `created_at` (DateTime) - Timestamp of interaction occurrence.
*   `dissolved_at` (DateTime, Nullable) - Expiry of temporal relationship.

### **FIRs**
First Information Reports (raw and parsed evidence).
*   `fir_id` (Primary Key, VarChar, 64)
*   `case_id` (VarChar, 64)
*   `district` (VarChar, 100)
*   `police_station` (VarChar, 100)
*   `date_filed` (DateTime)
*   `narrative` (VarChar, Max) - Cleaned text payload.
*   `sections` (VarChar, 255) - Comma-separated list of applicable legal sections.
*   `forensic_hash` (VarChar, 64) - SHA-256 hash of original raw voice/PDF evidence.

### **AI_Execution_Logs**
Forensic audit log for legal admissibility.
*   `log_id` (Primary Key, BigInt)
*   `correlation_id` (VarChar, 64)
*   `job_id` (VarChar, 64)
*   `timestamp` (DateTime)
*   `action` (VarChar, 100)
*   `prompt_tokens` (Int)
*   `completion_tokens` (Int)
*   `input_hash` (VarChar, 64) - Hash of raw request payload.
*   `signature` (VarChar, 64) - Hash chain linking to the previous record's signature.

## **2. IPC to BNS Ontological Mapping**

When queries reference legacy Indian Penal Code (IPC) sections, translate them automatically to the corresponding Bharatiya Nyaya Sanhita (BNS) sections to ensure multi-era case coverage:

| Offense | Legacy IPC Section | Modern BNS Section |
| :--- | :--- | :--- |
| **Murder** | IPC Section 302 | BNS Section 103 |
| **Attempt to Murder** | IPC Section 307 | BNS Section 109 |
| **Theft** | IPC Section 379 | BNS Section 303 |
| **Extortion** | IPC Section 384 | BNS Section 308 |
| **Cheating** | IPC Section 420 | BNS Section 318 |
| **Criminal Conspiracy** | IPC Section 120B | BNS Section 61(2) |
| **Unlawful Assembly** | IPC Section 143 | BNS Section 189 |
| **Hurt** | IPC Section 323 | BNS Section 115 |
| **Grievous Hurt** | IPC Section 325 | BNS Section 117 |
